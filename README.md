# Bag Recorder

A simple bag recording node. This allows for multi-bag recording, configurable topic recording, and most importantly, interrupt routines between Ctrl+C and bag stopping.

> [!NOTE]
> This implementation is originally based off of @VarCoder, @yaoyuh-cmu and @kabirkedia's work during their time in the DARPA Triage Challenge on [team Chiron](https://teamchiron.ai/)
